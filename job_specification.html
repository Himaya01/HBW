Job Specification

The job specification for a web developer position at Hot Beans include both technical qualifications and responsibilities

Qualifications:

  • Level 2 award in Creating a website
  • Skills in HTML, CSS and JavaScript
  • GCSE in IT
  • Level 3 Diploma in IT User Skills
  • GCSE Maths and English (Grade4 or above)

    • Responsibilities:

  • Developing user interfaces and ensuring responsive design for various devices.
  • Collaborating with designers to implement visual and interactive elements.
  • Maintaining and improving website performance.
  • Designing, building, and testing web applications.
  • pdating and maintaining websites, including content management and backups.
  • Identifying and resolving errors through testing and customer feedback
